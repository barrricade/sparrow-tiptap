# Install 📦

```shell
npm install --save sparrow-tiptap
```

# Example code (React)

```javascript
import Stiptap from 'sparrow-tiptap';
import 'sparrow-tiptap/core/style.css';
function App () {
  return (
    <>
      <Stiptap />
    </>
  )
}

export default App
```
